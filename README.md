# Interactive Calculator

# Description

This is an interactive, browser-based calculator application created with HTML, CSS and vanilla JavaScript as part of The Odin Project curriculum. It allows the user to do calculations using four basic arithmetic operations as well as root extraction and exponentiation. It also has keyboard support.

# Live Demo

[Live Demo](https://cumhurbabaoglu.github.io/odinproject-calculator/)

# Features
- Four standard arithmetic operations, e.g. addition, subtraction, multiplication and division
- Two extra functions including square root and exponentiation
- AC button to start a new calculation
- DEL button to delete the digit last entered
- Decimal point input
- Keyboard support

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
